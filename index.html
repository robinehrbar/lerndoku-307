Modul 307 - interaktive Webseiten

Einleitung

Wenn man die Grundlagen von HTML & CSS beherrscht, kann man sich an das, für Fortgeschrittene, PHP und JavaScript machen. Diese zwei Skriptsprachen bringen diverse Vorteile für die Interaktion mit Webseiten. Der wesentlichste Unterschied zwischen PHP und JavaScript ist, dass JavaScript auf dem Computer des Benutzers, mithilfe eines Browsers ausgeführt wird, wohingegen PHP auf einem Webserver läuft.

Arbeitsumgebung und Tools

Bevor man mit Skripten beginnt, sollte sichergestellt werden, dass man eine angenehme Arbeitsumgebung mit nützlichen Tools für sich vorbereitet. Essenzielle Tools wie diese dürfen nicht fehlen:

Editor
Visual Studio Code erleichtert das schreiben mit HTML, CSS, PHP und JS erheblich.
Browser
Google Chrome, Firefox, Opera, Internet Explorer (vorinstalliert bei Windows), Microsoft Edge (vorinstalliert bei Windows) und Safari (bei Apple), all diese Browser sollten beim Testen verwendet werden, da diese sehr häufig von Endbenutzern gebraucht werden.
Entwickler Tools
Chrome Development Tool ist äusserst nützlich beim Testen einer Webseite.
Webserver
XAMPP ermöglicht das Erstellen von einem lokalen Webserver.
(Grafiken)
Adobe Photoshop ist leider kostenpflichtig, ist aber auch nicht zwingend notwendig den GIMP kann das Wichtigste auch und ist bereits vorinstalliert bei neueren Windows-Versionen.
Wissen
Sehr empfehlenswerte Links sind einerseits w3schools und andererseits !selfhtml.

PHP erklärt

PHP ist die Abkürzung für Personal Home Page Tools. Ihre Syntax ähnelt der von C und Perl, sie dient bekanntlich zur Erstellung von dynamischen Webseiten und bei uns im Kurs hauptsächlich für Formulare. PHP ist nicht kostenpflichtig, sondern es kann gratis auf einem Webserver verwendet werden. Wie in der Einleitung schon erwähnt ist PHP serverbasiert, das heisst, dass alle Prozesse auf dem Webserver ablaufen und somit die Clients weniger effektive Rechenleistung auf ihrem Gerät vorweisen müssen.
Ursprünglich war PHP nur eine Ansammlung von Perl-Skripten, die Rasmus Lerdorf 1995 entwickelt hat, um seinen Online-Lebenslauf zu schreiben. Heute gilt dieser als Entwickler von PHP.

Aufbau Befehl (Syntax)

Wie eben erwähnt ist PHP aus Perl entstanden, entsprechend sieht auch ihre Syntax aus. Man startet ein PHP Skript immer mit einem <?php Tag und beendet ein Skript dann wieder mit folgendem Tag: ?>
Ein konkretes Beispiel zur Erstellung einer Funktion würde so aussehen: <?php
function frame() {
echo "Hello World";
}
?>
Um diese Funktion dann später zu benutzen, muss man folgende Syntax verwenden: <?php
frame();
?>
